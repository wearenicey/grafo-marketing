A table ili sendvič table

A table se izrađuju od eloksiranog aluminijuma koji se otvara i zatvara po principu klik-klak sistema. Štampa je zaštićena anti-refleks folijom čime joj se produžava vek trajanja u smisliu uv zaštite i vodo-otpornosti. Odlikuje ih izuzetna stabilnost, laka zamena štampe koja se obično radi na plakat papriu, mobilnost, laka montaža-demontaža.

Aluminijumske table

Aluminijumske table se izrađuju na podlozi koja je od aluminijuma. Obično se cela površina table prelepljuje štampanom PVC folijom.

Aluminijumski klik-klak ramovi

Klik ramovi se izrađuju od kvalitetnog srebrno anodizovanog aluminijuma. Sastoje se od leđnog panela na koji se montira poster i zaštitnog providnog panela koji se postavlja preko postera (štiti poster od vlage, prašine i drugih spoljnih uticaja). Izmena postera se vrši jednostavnim otvaranjem pokretnih lajsni i uklanjanjem zaštitnog panela. Aluminijumski ramovi se prodaju u kompletu sa anti-refleks folijom, paketom tiplova i šrafova.

Metalne table

Metalne table se izrađuju od metalne konstrukcije sa displejom od pocinkovanog lima. Postoji i kombinacija metalne konstrukcije i ceradnog platna. Metalne table su najbolje rešenje za putokaze, gradilišne table i banderašice.

Table od foreksa

Foreks table se izrađuju tako što se na podlozi koja je od foreksa lepi štampana PVC folija. Može se prelepiti cela površina foreksa ili se folija seče u određenim detaljima koji se lepe na podlogu. Takođe, postoji mogućnost direktne digitalne UV štampe koja je veoma postojana i otporna na grebanje.

Table od graverske plastike

Table od graverske plastike se koriste za označavanje poslovnih objekata, ulaznih vrata, službenih prostorija, hotela, ugostiteljskih objekata, advokatskih kancelarija, lekarskih ordinacija i dr. Materijal koji se koristi za izradu tabli je graverska plastika ili graverska folija koja je uv stabilna i otporna na grebanje. Njena standardna debljina je 1,5 mm.Table se personalizuju laserskom gravurom što isključuje svaku mogućnost da dođe do određenog skidanja boje ili promene boje slova

Table od kartonske pene

Table se izrađuju na podlozi koja je od kartonske pene. Kartonska pena (foam) je izuzetno lagan materijal koji je osetljiv na vlagu, pa se samim tim koristi samo za unutrašnju dekoraciju. Štampa se radi na PVC foliji, a zatim se table montiraju na zid uz pomoć duplofan trake ili silikona.

Table od klirita

Table od klirita ili pleksiglasa se izrađuju tako što se na podlozi koja je od klirita lepi štampana PVC folija. Može se prelepiti cela površina klirita ili se folija seče u određenim detaljima koji se lepe na podlogu. Takođe, postoji mogućnost direktne digitalne UV štampe koja je veoma postojana i otporna na grebanje.

Table od leksana

Table se izrađuju na podlozi od leksana. Leksan je lagan materijal i ima saćastu strukturu. Može biti u providnoj ili mlečnoj boji. Štampa se radi na PVC foliji. Može se prelepiti cela površina leksana ili se folija seče u određenim detaljima koji se lepe na podlogu. Takođe, postoji mogućnost direktne digitalne UV štampe koja je veoma postojana i otporna na grebanje.

Reklamne table

  • Opis
  • Vrste

Reklamne table su idealan vid oglašavanja. Čine savršen proizvod za sve koji žele da njihov brend bude u potpunosti vidljiv i prepoznatljiv. Table se rade na svim vrstama podloga i različitim tehnikama brendiranja. Mogu biti namenjene za unutrašnje i spoljašnje obeležavanje ili reklamu.
